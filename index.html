What is hardware?

  • Hardware is real, physical, tangible stuff that you can see and touch.
    •
  • It's the phone in your pocket, the laptop in your bag, or the pile of components that we are about to work with.
    •

What is software?

  • Software is the rules and instructions that tell a computer what to do.
    •
  • It's the App on your phone, the web browser on your laptop, desktop, tablet, TV, or anything else.
    •
  • It is any program that runs on hardware: Android, iOS, Windows, MacOS, Linux, Chrome, Firefox, Fortnite
    •

What is firmware?

  • Firmware sits in the middle. It's the code that helps hardware and software work effectively together.
    •
  • The processor, graphics card, mainboard/motherboard, hard drives, SSDs, even MicroSD cards all have firmware.
    •
  • Firmware helps software make hardware work.
    •

Expectations

  • Our first goal in this lab is safety: We will not run, toss or throw things, or be abusive.
    •
  • Our second expectation in the lab is kindness. We will be respectful, kind, and polite.

  • We will respect every person and every item in this lab.
    •

Safety & Precautions

  • Computer components are susceptible to damage from misuse, drops, and static electricity.
    (Touching a large metal object can help discharge static electricity.)
    •
  • Most components are physically keyed to prevent them from being installed improperly
    •
  • If you do not drop components, use excessive force to install them, and regularly discharge static, you will not have a problem
    •

Computer Case

Computer Case

Houses and protects all components
Provides routing for both power and data cables
Mounts fans, drives, cards, and speaker

Routes airflow

Motherboard/Mainboard

Motherboard

Interconnect between CPU, RAM, drives, cards, I/O ports and other functions
Contains many devices necessary for the computer to function

I/O Shield

I/O Shield

Allows access to the motherboard ports
Protects from dust intrusion

Random Access Memory

RAM

Stores data that the processor requires for programs to run or information to be accessed
Short-term, volatile

Central Processing Unit

CPU

Highly complex electronic circuitry
Runs programs, arithmetic, & logic
Controls data movement, manages much of the system I/O

Graphical Processing Unit

GPU

Digital image processing, accelerates graphics, and outputs to a display
Can also be used for processing massive amounts of parallel data for research, AI, and other mathematical operations

Power Supply Unit

PSU

Converts AC power to DC and distributes it to system components
Provides fault protection from both the input and output sides

CPU Heat Sink and Fan

CPU Heat Sink and Fan

Provides a central, conductive mass to absorb waste heat from the CPU
Has a significant amount of surface area over which the fan moves air to dissipate heat

Thermal Paste

Thermal Paste

Thermally conductive, electrically insulative
Eliminates air gaps from imperfect materials ensuring uniform heat transfer

Displays

Displays

Displays visual content such as text, images, or other information produced by a computer
Come in a large variety of sizes, types, technologies, and form factors; all have the same basic purpose
